# Shanthi Akula — Hypnotherapy & Wellness Website

Premium, SEO-first website for **Shanthi Akula**, Consultant Psychologist & Healing Therapist (Hyderabad).
Replaces the old `shanthiakula.graphy.com`.

## Structure
- `demo/` — static demo site (Home, About, Services, Hypnotherapy detail, Blog) — deploy target for GitHub Pages
- `docs/SYSTEM_OVERVIEW.md` — single source of truth / audit diary (requirements, decisions, change log, tests)
- `docs/CLIENT_QUESTIONS.md` — client intake questionnaire
- `assets/` — organised photos & credentials (`portraits/`, `events/`, `credentials/`)

## Demo site
Plain HTML/CSS, no build step. Open `demo/index.html` locally, or serve the `demo/` folder.

> This is an approval preview. Content, pricing, booking, domain and final assets are pending client sign-off.
